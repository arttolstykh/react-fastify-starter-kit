# Custom project

Full-stack Boilerplate

## Install:

```bash
npm install
```

## Running:

### Production

```bash
npm start
```

### Development

```
npm run develop
```

### Front-end Development

```
npm run start:client
```

Listen: `http://localhost:4000`

### Back-end Development

```
npm run start:server
```

Listen: `http://localhost:3000`

### Test

```
npm test
```
